# Chesemchanel - Week 02 Project

A fully responsive, styled portfolio website built during Week 2: CSS Mastery. This week focused on transforming an HTML portfolio into a beautifully styled, responsive website using modern CSS techniques.

## Live Demo

[View Live Site](https://chesemchanel.github.io/iyf-s10-week-02-chesemchanel)

## Screenshot

![Project Screenshot](./images/screenshot.png)

## Features

- ✅ Responsive design (mobile-first)
- ✅ CSS custom properties (variables)
- ✅ Typography system with font pairing
- ✅ Color scheme system
- ✅ Flexbox navigation bar
- ✅ CSS Grid photo gallery & project grid
- ✅ Box model practice page
- ✅ Hover and focus states
- ✅ Smooth transitions
- ✅ Accessible (WCAG compliant focus styles)
- ✅ Mobile-friendly contact form
- ✅ Multi-column footer

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Google Fonts (Poppins + Open Sans)
- Mobile-first responsive design

## What I Learned

- Setting up a CSS file and linking it to HTML pages
- Creating a typography system with CSS variables
- Building a color scheme with CSS custom properties
- Using Flexbox for navigation and card layouts
- Using CSS Grid for photo galleries and page layouts
- Making sites responsive with media queries (mobile-first)
- Styling forms with focus and error states
- Adding hover states and smooth transitions
